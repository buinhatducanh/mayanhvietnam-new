## 🏆 Top 10 — Design Systems & UI Libraries

### 1. MUI (Material UI) ⭐ 118k — [mui/material-ui](https://github.com/mui/material-ui)
- **License:** MIT
- **Stack:** React + TypeScript
- **Mô tả:** Library React phổ biến nhất thế giới, hiện thực Material Design 3. Có MUI Core (free) và MUI X (advanced components: DataGrid, DatePicker, Charts).
- **Phù hợp mayanhvietnam?** ✅ Cực tốt — **ĐÃ CÓ TRONG demo-02** (`@mui/material 7.3.5`). Component phong phú, e-commerce templates sẵn, theme customize cho brand camera.

### 2. Ant Design ⭐ 98.6k — [ant-design/ant-design](https://github.com/ant-design/ant-design)
- **License:** MIT
- **Stack:** React + TypeScript
- **Mô tả:** Library enterprise-level, design system hoàn chỉnh từ Anthropic. Có version Vue (ant-design-vue) và mobile (antd-mobile).
- **Phù hợp mayanhvietnam?** ✅ Rất tốt cho **admin-next** — components bảng, form, filter phong phú, dễ build CMS quản lý sản phẩm camera.

### 3. Mantine ⭐ 98.6k — [mantinedev/mantine](https://github.com/mantinedev/mantine)
- **License:** MIT
- **Stack:** React + TypeScript
- **Mô tả:** Full-featured component library, có hooks riêng (`@mantine/hooks`), form management, notifications, charts, date picker, modals, color picker — tất cả trong 1.
- **Phù hợp mayanhvietnam?** ✅ Tốt — bundle nhỏ, DX cao, theme sạch. Thay thế tốt cho MUI nếu muốn nhẹ hơn.

### 4. Chakra UI ⭐ 41.4k — [chakra-ui/chakra-ui](https://github.com/chakra-ui/chakra-ui)
- **License:** MIT
- **Stack:** React + TypeScript
- **Mô tả:** Focus vào accessibility + DX, có v3 mới (2024). Composability tốt, dễ customize theme.
- **Phù hợp mayanhvietnam?** ⚠️ Trung bình — đang mất dần momentum so với shadcn/ui. Dùng cho prototype thì OK.

### 5. Element Plus ⭐ 41k — [element-plus/element-plus](https://github.com/element-plus/element-plus)
- **License:** MIT
- **Stack:** Vue 3
- **Mô tả:** Vue 3 port của Element UI. Phổ biến ở Trung Quốc.
- **Phù hợp mayanhvietnam?** ❌ Không phù hợp — dự án đang dùng React.

### 6. DaisyUI ⭐ 40.5k — [saadeghi/daisyui](https://github.com/saadeghi/daisyui)
- **License:** MIT
- **Stack:** Tailwind plugin
- **Mô tả:** Component classes cho Tailwind CSS, không cần JS runtime. Nhẹ, dễ kết hợp với bất kỳ framework nào.
- **Phù hợp mayanhvietnam?** ✅ Rất tốt — bạn đang dùng Tailwind 4, daisyUI có thể plug-in ngay để giảm code.

### 7. shadcn/ui ⭐ 31.4k — [shadcn-ui/ui](https://github.com/shadcn-ui/ui)
- **License:** MIT
- **Stack:** React + Radix UI + Tailwind CSS
- **Mô tả:** **KHÔNG phải package** — đây là bộ component **copy-paste** (CLI: `npx shadcn@latest add button`). Components dùng Radix UI primitives + Tailwind CSS. Trending #1 trong 2024–2025.
- **Phù hợp mayanhvietnam?** ✅ **Rất phù hợp** — bạn đã dùng `@radix-ui/*` (26 packages trong demo-02). Có thể adopt shadcn pattern để giảm dependency.

### 8. HeroUI (NextUI) ⭐ 29.8k — [heroui-inc/heroui](https://github.com/heroui-inc/heroui)
- **License:** MIT
- **Stack:** React + TypeScript
- **Mô tả:** Modern UI library, modern design aesthetic. Đổi tên từ NextUI → HeroUI.
- **Phù hợp mayanhvietnam?** ✅ Tốt — visual đẹp, animation mượt.

### 9. Adobe React Spectrum ⭐ 27.6k — [adobe/react-spectrum](https://github.com/adobe/react-spectrum)
- **License:** Apache-2.0
- **Stack:** React + TypeScript
- **Mô tả:** Design system chính thức của Adobe (Photoshop, Illustrator, Premiere). Focus accessibility tuyệt đối (WCAG 2.1 AAA).
- **Phù hợp mayanhvietnam?** ⚠️ Bundle lớn, design language riêng của Adobe. Dùng cho admin tools nhiều keyboard-shortcut thì tốt.

### 10. Magic UI ⭐ 27.3k — [magicuidesign/magicui](https://github.com/magicuidesign/magicui)
- **License:** MIT
- **Stack:** React + Tailwind + Framer Motion
- **Mô tả:** Bộ **animated components** (buttons, text effects, cards, marquees) dùng Framer Motion. Trending 2024–2025 cho landing page đẹp.
- **Phù hợp mayanhvietnam?** ✅ **Rất phù hợp** — bạn đã có `framer-motion 12.42.2` trong demo-01. Magic UI giúp landing page "wow".

## 🎨 Design Tools & Platforms

### 11. Tldraw ⭐ 48.6k — [tldraw/tldraw](https://github.com/tldraw/tldraw)

- **Phù hợp mayanhvietnam?** ✅ Dùng làm design tool nội bộ cho team — free, không giới hạn seats, không vendor lock-in.

## 🎯 Gợi ý cho dự án mayanhvietnam

### Dựa trên stack hiện tại của bạn:

```
✅ Đang dùng:        Next.js 16 + React 19 + Tailwind 4 + Radix UI + MUI 7 + Recharts + Framer Motion + Lucide
```

### Khuyến nghị adopt:

| Repo | Lý do | Effort |
|------|-------|--------|
| **[shadcn/ui](https://github.com/shadcn-ui/ui)** | Đã có Radix UI + Tailwind 4 sẵn → copy-paste pattern để giảm bundle, dễ customize brand camera | Trung bình |
| **[daisyUI](https://github.com/saadeghi/daisyui)** | Tailwind plugin thuần → giảm component code cho demo-02 đang có 60+ deps | Thấp |
| **[Magic UI](https://github.com/magicuidesign/magicui)** | Đã có `framer-motion 12.42.2` → landing page hero/banner/feature section đẹp | Thấp |
| **[Penpot](https://github.com/penpot/penpot)** | Self-host làm design tool nội bộ cho team, thay Figma | Cao (deploy) |
| **[Tremor](https://github.com/tremorlabs/tremor)** | Thay vì custom MUI dashboard cho admin-next, dùng Tremor component | Trung bình |

### Anti-recommendations (đừng thêm vào):
- **Bootstrap** — overlap 100% với Tailwind
- **Vuetify, Element Plus, PrimeNG** — sai framework (Vue/Angular)
- **Chakra UI** — đang mất momentum
