## ℹ️ Ghi chú kỹ thuật

| Thuộc tính | Giá trị |
|------------|---------|
| **Format ảnh sản phẩm** | JPG为主, JPEG cho đèn Godox, Webp cho category/banner |
| **Kích thước avatar** | ~500x500px (thumbnail), một số file lớn ~2000x2000px |
| **CORS** | Public CDN — load trực tiếp trên trình duyệt, không cần proxy |
| **Auth** | Không cần — tất cả ảnh đều public |
| **Tổng số unique product images** | **~230+** (across all categories) |
| **Pattern URL sản phẩm** | `/image-data/san-pham/{YY-MM}/{YY-MM-DD}/{timestamp}/avatar/{id}_{slug}.jpg` |
| **Pattern URL asset** | `/asset/imgs/{icon\|img}/{filename}` |
